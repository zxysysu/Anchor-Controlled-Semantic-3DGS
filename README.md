## Data Preparation

First, create a `data/` folder inside the project path:

```bash
mkdir data
```
The data structure should be organized as follows:
```bash
data/
├── dataset_name
│   ├── scene1/
│   │   ├── images
│   │   │   ├── IMG_0.jpg
│   │   │   ├── IMG_1.jpg
│   │   │   ├── ...
│   │   ├── sparse/
│   │   │   └── 0/
│   │   ├── segment/
│   │   │   ├── IMG_0_color.jpg
│   │   ├── segment_pred/
│   │   │   ├── IMG_0_color.jpg
```
